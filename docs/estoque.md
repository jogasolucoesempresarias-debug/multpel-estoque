- `INFO.TABLES()` / `INFO.COLUMNS()` → retornam **erro 400** nesse tenant. Usar as variantes **`INFO.VIEW.TABLES()`** / **`INFO.VIEW.COLUMNS()`** (colunas úteis: `[Name]`, `[Table]`).
- Cada modelo carrega dezenas de `LocalDateTable_*` / `DateTableTemplate_*` automáticas — **ruído**, ignorar.

- O dataset **Painel_Gerencial_Comercial** existe mas o Service Principal **não tem permissão** (401) — se precisar dele depois, liberar acesso.
